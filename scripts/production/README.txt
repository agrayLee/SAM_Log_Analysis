Sam Log System - Production Scripts 
======================================= 
 
This folder contains all scripts required for production deployment. 
 
USAGE: 
1. Copy all files in this folder to target deployment server 
2. Run one-click-deploy.bat as administrator 
3. Script will automatically choose best deployment version 
 
PRIORITY ORDER: 
1. migration-deploy-clean.bat (preferred - npm issue fixed) 
2. migration-deploy-simple-en.bat (backup - English version) 
3. migration-deploy-simple.bat (last resort - Chinese version)
